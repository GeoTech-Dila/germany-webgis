## 🚀 Proje Özeti

Bu proje, coğrafi verilerin web ortamında görselleştirilmesi ve analiz edilmesi amacıyla geliştirilmiştir.

---
              
## 🧩 Özellikler

### 🗺️ Harita (Layers)

* Eyalet bazlı renklendirme
* Opacity (saydamlık) ayarı
* Renk değiştirme
* Tıklayınca zoom

---

## 📊 Analiz Paneli

* Alan bazlı sıralama
* Animasyonlu analiz
* Line chart + Donut chart
* Harita ile senkron animasyon

---

## 📐 Ölçüm Araçları

* Polygon → Alan (hektar)
* Polyline → Uzunluk (metre)
* Harita üzerinde interaktif ölçüm

---

## 🛠️ Kullanılan Teknolojiler

* Leaflet.js
* Chart.js
* JavaScript
* GeoJSON
* Vercel

---

## 📦 Veri Kaynağı

* Almanya eyalet sınırları (GeoJSON)
* GeoPandas ile hesaplanmış alan verileri

---

## 🧠 Analiz Metodolojisi

Bu projede kullanılan eyalet alan verileri, web ortamında değil veri hazırlama sürecinde hesaplanmıştır.
